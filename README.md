## General info

The sponsor has given the responsibilities to Team 10 to create the project known as “Huntr”. The project requires Team 10 to create a web-based system where contractors can post jobs and search for potential trades personal based on skills that match the description provided by the contractor and for trades personal to search for jobs using a search system that would fit their skills. The project system will cover all the requirements listed below. 

- Permit users to register [Admin/employer/employee]. 
- Provide necessary transactions for data entry. 
- Provide match making algorithms for users to search. 
- Have Appropriate admin functionality [Edit/Delete].

## How to run

### Download files
- Download/Clone Front-End files from the master branch.
- Download/Clone Back-end files from Timal/Backend branch.

### Install and Run

- Open Front end and backend in seperate VS Code Environments.
- Use ``` npm install ``` to install front end and back end.
- Use ``` npm run ``` to run both environments together.
